Hashmlai
An electrical business management system, which consists of Internet application and smartphone application, 
using Python, Django, React,React native and Mysqlite. 
Users can automatically extract a quotation from wiring diagrams using manual labeling, easy access functionality,
and advanced algorithms.



To run:
Install npm, react native on your machine.
Install Xcode and android studio, then activate the simulatora.
enter the main directory from terminal and type:
react-nativbe run-ios - in case youi want to run on iphone
react-native run-android - in case you want to run on android studio.
all is well the application have to run smoothly:)
